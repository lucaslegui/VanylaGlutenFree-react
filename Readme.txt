# VanylaCakes - Productos Gluten Free

Este proyecto consiste en una aplicación para la gestión y visualización de productos sin gluten de la pastelería VanylaCakes. La aplicación está dividida en dos partes principales: la API backend y el frontend.

## Autor

**Lucas Leguizamón**

## Descripción del Proyecto

La API backend proporciona los endpoints necesarios para gestionar los productos, incluyendo la autenticación y la subida de imágenes. El frontend está desarrollado con React y proporciona una interfaz de usuario para interactuar con la API y mostrar los productos de manera atractiva.

## Tecnologías Utilizadas

### Backend

El backend está construido utilizando Node.js y Express, con una base de datos MongoDB. Las principales dependencias utilizadas son:

- **axios**: Para realizar solicitudes HTTP desde Node.js.
- **bcryptjs**: Para el hashing de contraseñas.
- **cors**: Para permitir solicitudes entre diferentes dominios.
- **dotenv**: Para manejar variables de entorno.
- **express**: Un framework para construir aplicaciones web y APIs.
- **jsonwebtoken**: Para la generación y verificación de tokens JWT.
- **mongoose**: Para la modelación de datos en MongoDB.
- **multer**: Para la subida de archivos.
- **nodemon**: Para el reinicio automático del servidor durante el desarrollo.

### Frontend

El frontend está desarrollado utilizando React y Vite para una experiencia de desarrollo rápida y moderna. Las principales dependencias y herramientas utilizadas son:

- **@popperjs/core**: Para la gestión de poppers (elementos emergentes).
- **axios**: Para realizar solicitudes HTTP desde el frontend.
- **bootstrap**: Para el diseño y la creación de componentes responsivos.
- **js-cookie**: Para manejar cookies en el navegador.
- **jwt-decode**: Para decodificar tokens JWT.
- **react-toastify**: Para la creación de notificaciones.

Las herramientas de desarrollo utilizadas incluyen:

- **@types/react** y **@types/react-dom**: Tipos para React.
- **@vitejs/plugin-react**: Plugin de Vite para React.
- **eslint**: Herramienta para identificar y reportar patrones encontrados en el código ECMAScript/JavaScript.
- **eslint-plugin-react** y **eslint-plugin-react-hooks**: Plugins de ESLint para React.
- **vite**

## Instalación y Uso

### Backend

1. Clonar el repositorio del backend.
2. Navega al directorio del proyecto.
3. Instala las dependencias utilizando `npm install`.
4. Crea un archivo `.env` y define tus variables de entorno.
5. Inicia el servidor con `npm start`.

### Frontend

1. Clona el repositorio del frontend.
2. Navega al directorio del proyecto.
3. Instala las dependencias utilizando `npm install`.
4. Inicia la aplicación con `npm run dev`.
